N-Queen Problem using Backtracking

Definition:
Place N queens on an N×N chessboard such that no two queens attack each other.

Concept Used:

- Backtracking
- Constraint Satisfaction

Conditions:

- No two queens in same column
- No two queens in same diagonal

Pseudocode:

isSafe(board, row, col, n):
    for i = 0 to row-1:
        if board[i][col] == 1:
            return false

    for i=row-1, j=col-1 while i>=0 and j>=0:
        if board[i][j] == 1:
            return false

    for i=row-1, j=col+1 while i>=0 and j<n:
        if board[i][j] == 1:
            return false

    return true

Space Complexity: O(N²)
